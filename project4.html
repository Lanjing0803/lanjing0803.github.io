About the Project

We’re Not Really Strangers is a brand that sells card games meant for people to get to know each other better on different levels (friendships, relationships, etc). The intention behind the brand is for players to forge meaningful connections with others.
To improve the user experience(UX) on this website, our Usability Study Group utilized Unmoderated Remote User Testing to collect feedback from participants and generate recommendations. This method involves the use of a software application to conduct studies, eliminating the need for a researcher to be present during the sessions. As a result, we formulated valuable recommendations for enhancing the website's overall user experience.
